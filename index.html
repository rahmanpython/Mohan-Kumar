Smart Recycling Bin Using Arduino

The "Smart Recycling Bin Using Arduino" project creates an automated system to sort recyclable and non-recyclable waste, enhancing recycling efficiency. Using IR, Arduino mega, Inductive proximity sensor and Capacitive proximity sensor. The bin detects and classifies waste materials. Sensors analyze recyclability upon waste deposit, and a motorized mechanism directs it to the correct compartment. This system promotes responsible waste disposal and environmental sustainability.
